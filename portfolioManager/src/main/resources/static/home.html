Wellcome to Stokker - Portfolio Manager

Please log in using your GitHub account to access the application.

This is a small personal learning project that intends offer some Portfolio Management functionality as well as some Stock price analysis.

  • You have more information about the project in the wiki along with instructions on how to lunch all the services.

Technology stack

  • AngularJS in the front-end.
  • Spring Boot 1.3.0 in the Server Side.
  • JPA (H2) and ElasticSearch as data persistence layer.
  • Kibana helps with the data visualization.
  • Some Docker containers are used to launch the services.

The source code is available in this GitHub repository.

  • Feel free to fork it, however I am still cleaning up things (like the commit policy, there is neither CI nor automatic testing - for the moment)
  • My objective is to turn this sandbox repository in something closer to an open-source project

You can follow the development of this project in my blog (http://victorferrerjava.blogspot.com.es/) along with some other software-related articles.
